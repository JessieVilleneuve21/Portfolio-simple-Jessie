Site web restaurant

Site web restaurant, La table d'olivier

Conception et développement d’un site web pour le restaurant La Table d’Olivier, réalisé en collaboration dans le cadre d’un projet académique.
Le design visuel et l’interface utilisateur ont été conçus par un coéquipier artiste, tandis que j’ai assuré le développement fonctionnel du site.
Implémentation de deux systèmes CRUD :

- un système de gestion des réservations avec calendrier interactif,
- un système de gestion des menus, incluant une cave à vin administrable.
- Le projet met en valeur la gestion dynamique de contenu, la structuration des données et l’expérience utilisateur.
